# 微考勤（WeAttandance）
#### 微考勤系统［以下简称WAM系统]
#### 本项目适用于会议考勤，课堂考勤等考勤方式。
#### 通过WAM系统，管理者可以统计出席人员，迟到人员，缺席人员。信息化管理每次考勤。
